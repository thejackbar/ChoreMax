Critical Priority — Fix Immediately +18 points

1 Fix Double-Slash Canonical URLs (All 50 Pages)
Every page has a canonical like https://maxgeo.com//exploration. This is caused by metadataBase in your Next.js root layout having a trailing slash.

Fix: In app/layout.tsx, change:
metadataBase: new URL('https://maxgeo.com/') → metadataBase: new URL('https://maxgeo.com')

This single character change fixes all 50 canonical URLs instantly.
Impact: +6 pts Effort: 5 min File: app/layout.tsx
2 Remove 5 Dead URLs from Sitemap
These sitemap URLs return 404: /datashed-training, /assay-management, /drilling-management, /logchief-training, /logchief-lite-training.

Fix: Exclude these paths from your sitemap generation. If using a custom sitemap.ts route, filter them out. Also add 301 redirects:
• /datashed-training → /training
• /logchief-training → /training
• /logchief-lite-training → /training
• /assay-management → /exploration
• /drilling-management → /exploration
Impact: +3 pts Effort: 15 min File: sitemap.ts + next.config.js (redirects)
3 Add Meta Descriptions to Homepage & Core Pages
The homepage, /exploration, /grade-control, /training, and /legal are all missing meta descriptions. These are high-traffic entry points.

Fix: Add descriptions via Sanity CMS metadata fields (or hardcode in page-level generateMetadata()). Aim for 120-160 characters with primary keywords. Refer to the meta content document for optimised descriptions.
Impact: +3 pts Effort: 30 min Where: Sanity CMS or app/*/page.tsx
4 Fix Hero Image loading="lazy" (LCP Killer)
The homepage hero image (1920x1080 JPG) uses loading="lazy" which delays the LCP element by 500-2000ms.

Fix: Add priority prop to the hero's Next.js <Image> component. This automatically sets loading="eager" and fetchpriority="high".
Impact: +3 pts Effort: 5 min File: Hero component
5 Fix HSTS max-age=0
Strict-Transport-Security: max-age=0 effectively disables HSTS. Browsers will not enforce HTTPS on subsequent visits.

Fix: In Cloudflare dashboard → SSL/TLS → Edge Certificates → enable "Always Use HTTPS" and set HSTS to max-age=31536000; includeSubDomains; preload.
Impact: +2 pts Effort: 5 min Where: Cloudflare Dashboard
6 Add 3 Software Pages to Sitemap
/software/datashed, /software/logchief, and /software/logchief-lite all return 200 but are NOT in the sitemap. These are key product pages.

Fix: Add these URLs to your sitemap generation logic with high priority.
Impact: +1 pt Effort: 10 min File: sitemap.ts
